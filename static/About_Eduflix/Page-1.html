About EduFlix

EduFlix

EduFlix is an E-learning management system with web services oriented framework. This system supports the cross browser. This system focuses around the several features namely Content Management, Content Protection, Learning Management, Delivery Management, Evaluation management, Access Control and mainly focuses on integrated platform needed for e-learning and managements. The application is designed and focused for students to help in their Learning process. 

Eduflix

This Minor Project Edu-Flix is an online educational Platform for learning. Eduflix runs on Desktops Laptops and Mobile Phones.
